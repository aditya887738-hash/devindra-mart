```bash
npm install
npm run dev
```

- `/` Customer App
- `/admin` Admin App
- `/billing` Billing/Merchant App
- `/rider` Rider App
- `/about` About Page

- 4 separate apps connected by role routing

- Firestore rules

- WhatsApp order only for Devindra Mart Wholesale

- 7 day pickup code no-repeat helper
- Multi-language engine
- Retail/Wholesale rules
- Khata, delivery, settlement, COD engines
